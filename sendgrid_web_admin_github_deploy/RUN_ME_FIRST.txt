1. Double click install_all_safe.bat
2. Edit .env:
   ADMIN_PASSWORD
   SECRET_KEY
   SERVICE_TOKEN
3. Double click start_web_admin.bat
4. Open http://127.0.0.1:8080
5. Login as admin, then open 团队成员 to add team members.
6. Open 收件人池/库 and upload one or multiple TXT/CSV recipient files into:
   - 0-3天库 for day 1, day 2 and day 3
   - 4-30天库 for day 4 through day 30
7. Create a task. You no longer select recipient lists in the task form.
8. Click 生成计划. The system automatically takes recipients from the two pools.
9. If the pool is insufficient, the page will show a popup alert and will not create the plan.

Warm-up speed:
Day 1 <= 60, actual 45-60
Day 2 <= 200, actual 185-200
Day 3 <= 700, actual 685-700
Day 4-30 <= 1000/day, actual 985-1000/day

Recipient pool upload supports multi-file selection and no longer uses MAX_RECIPIENT_UPLOAD_BYTES.
